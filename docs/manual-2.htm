2. The Game

2.1 Concept

The concept is a platform deathmatch game. You run around a tile-based map, picking up various items and shooting your opponent. Original.

2.2 Story

(ahem)

You have been thrown in a dungeon with a handful of other prisoners. Contained within the dungeon are various magical objects, mostly weapons, which respawn after a certain time period.

Surrounded by such weapons of destruction, you cannot resist the temptation...

2.3 Starting the game

Simply run the executable red.exe (or red under Linux) to start. Please note the following command-line options are case-sensitive:

--edit, -e

Start semi-built-in map editor.

--lcd, -l

Use bright (ugly) version of mouse cursor. Designed to be used on hard to see laptop displays.

--stats, -s

Use alternative stats for items.

--com1, -1

Use COM1 for serial connections.

--com[234], -2, -3, -4

Ditto. Default is 2.

--skip, -s

Skip introduction.

--version, -v

Display version number.

Note that you cannot collate the options. i.e. red -1ql is invalid.

Linux users: COM1 = /dev/ttyS0; COM2 = /dev/ttyS1; etc.

2.4 Serial connection details

Link two computers via a serial cable in COM 1-4 (see Command line options in section Starting the game). Select `Start' --> `Serial'. Enter your name, then wait for the computers to connect. If it hangs, press `Ctrl-Alt-End' and try again.

If it says `Error opening COM port', check to make sure you used the right --comX option. If you're are running Windows 9x, try running under pure DOS. If you have a Logitech mouse you could also also try what I did, and get updated mouse drivers (MouseWare) for it from their web site.

From here on, you will be shown a map selection screen. Only the maps existing on both computers will be displayed. Please make sure the maps are identical, otherwise your games will be unsynchronised. Press the up and down arrow keys to select a map. You may also have a conversation of here if you wish. Feel free to use any obscene words you desire.

(Note: The chat box was designed in case the game ever supported Internet play. However it may be also useful in other areas, e.g. library)

When you are ready to play, press F10.

If your game become unsynchronised, quit the level and restart. If it ever says SYNC PROBLEM, then you'd better restart from the serial connection procedure.

2.5 Controls

  • Use the following keys to move around. They are all equivalent. 
            W               T                       Up
    A - S - D       F - G - H           Left - Down - Right
  • Aim the gun with the mouse, fire with the left mouse button.
  • Select the weapon with the number keys, and with Q, E, R, Y. The weapons you have collected will be displayed on the left. Press Enter or press the right mouse button to lay a mine.
  • TAB to see your scores. ESC to quit.
  • F12 to see the frame rate counter. This is a development feature, so it's not supposed to work properly. Just push it quickly and see.

2.6 Items

  • Weapons
    • Knife
    • Pistol
    • Shotgun
    • Uzi
    • M16 (just a big gun, nothing like an M16)
    • Minigun
    • Bow with explosive arrows
    • Bazooka
    • Mines
  • Other items:
    • Chicken (+health)
    • Coke (+health)
    • Armour. Takes half the damage from an attack (1 health = 2 armour)
    • Goggles. Lights up your vision, for about a minute.
    • Backpack. You drop half of all your ammo when you die, but your guns disappear.
    • Bloodlust. Double damage for all weapons, except mines. Be careful! Lasts 30 seconds.
    • Scanner. Points in the direction of your enemy.

2.7 Stats

Included with the game are three files stats, stats.07 stats.fst. stats is the more up to date version, providing more powerful weapons with quicker reload and respawn rates. stats.07 contains the older values, kept only for historic reasons. stats.fst contains values that will quickly increase the body count.

You can create your own stats by copying one of them, and editing them with a text editor, then use it with 

> red --stats mystats
Note that both machines will need exactly the same file, and both must run with the -s command-line option. You have been warned.

2.8 Tips

  • Use the 3D sound to find where your opponent is. Also, eating food does not make a noise, but picking up things does.
  • After you pick up an item, place a mine there. Works also for regenerating walls (will not flash).
  • The scanner is good for pursuit, as well as flight.
  • If are being mowed down with a minigun, run and drop mines! The blood will conceal it :)
  • Jump over your enemy and lay mines.
  • Jump up ladders, never climb.
