3. The Map Editor

3.1 Map editor keys

After starting the map editor with red --edit you will be presented with a blank screen. On the bottom there is a palette and on the top there is a description of the current map, showing the top-left map coordinates and size.

  • arrow keys to pan around
  • 1, 2, 3 to change editing modes (tiles, items, starting locations)
  • I, J, K, L to change map size
  • Ctrl-N to clear
  • Ctrl-S to save
  • Ctrl-L to load
  • Ctrl-Q to quit (warning: does not prompt!)

Select an item or tile with the left mouse button, place it on the map with the LMB. Right mouse button clears. That's it! There is a maximum of 24 starting locations on a single map.

3.2 Custom maps

If you wish to play your custom maps, make sure that you have exactly the same file on both machines! And make sure you send them to me so I can put them up on my web page.
